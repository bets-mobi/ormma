ORMMA Level One Sample Ad

Overview

This ad exercises all the functions of the Level1 specification. The JavaScript source is not minimized, so you can view it directly, http://ormma.googlecode.com/svn/trunk/AdSamples/OrmmaOneAd/OrmmaOne.js.

To run this ad in the web tester:

  • Use the URL, "http://ormma.google.com/svn/trunk/AdSamples/OrmmaOneAd/OrmmaOneFragment.html" and be sure to check the "fragment" option
  • Use the HTML tag, and paste in: 
    <link href="http://ormma.googlecode.com/svn/trunk/AdSamples/OrmmaOneAd/OrmmaOne.css" rel="stylesheet" type="text/css"/>
<script src="http://ormma.googlecode.com/svn/trunk/AdSamples/OrmmaOneAd/OrmmaOne.js" type="text/javascript" language="javascript"></script>
<div id="adSpace" style="height:50px">
  <div id='ad'>
    <div id='banner'><img src="http://media.crispadvertising.com/adframework/1.2/components/img/300x50.gif" alt="banner advertisement" onclick="return(expand());" /></div>
    <div id='closebanner'><img src="http://media.crispadvertising.com/adframework/1.2/components/img/btn_close.gif" alt="" onclick="return(hide());" /></div>
    <div id='panel'>
      <div id='background'><img src="http://media.crispadvertising.com/adframework/1.2/components/img/300x250.gif" alt="panel" /></div>
      <div id='hotspot'><a href="http://blog.ormma.org/" onclick="return(open(this.href));"><img src="http://ormma.googlecode.com/svn/trunk/AdSamples/OrmmaOneAd/small_ormma.gif" alt="ORMMA" /></a></div>
      <div id='close'><img src="http://media.crispadvertising.com/adframework/1.2/components/img/btn_close.gif" alt="close" onclick="return(collapse());" /></div>
      <div id='ormma'>Running in ORMMA compliant container.</div>
    </div>
  </div>
</div>

User testing

The expected behavior of this ad is to display the banner automatically, click to banner expand, click the X to collapse and click the X again to hide the ad. Clicking on the Facebook icon will open the http://ormma.org site.

This ad will work in a standard browser, although you will see comments in the JavaScript console that ORMMA is not found. The fragment ad which does not include these instructions will work in a standard browser, web tester, and ORMMA-compatible SDKs.
